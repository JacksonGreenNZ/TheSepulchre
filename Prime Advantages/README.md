# Prime Advantages

Prime Advantages: the upgrades units can take in Prime slots.

## Content type: set

Standalone advantage collections are a folder with `meta.yaml`, `README.md`, and `rules.pdf`. Copy `/_template/`.

Advantages tied to a specific detachment do not go here — they ship inside that detachment's PDF in `/detachments/` or `/restored-content/`.

## Compatibility

Additive. `replaces` stays empty; advantages stack unless two define the same-named advantage.

Small, single-advantage changes are better filed as tweaks in `/tweaks/` than as their own set. Use this folder for collections substantial enough to warrant a PDF.
